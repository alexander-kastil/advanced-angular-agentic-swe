Arrow Functions in Template Expressions (v21.2)

Write function logic directly in templates without wrapper methods:

Current count: {{ count() }}

Multiplier: {{ multiplier() }}

Doubled: {{ count() * multiplier() }}

(click)="count.update(n => n + 1)" - Arrow function in click handler

Spread Syntax in Templates (v21.1)

Use spread operator with objects and arrays in template expressions:

Original Products:

@for (product of products(); track product.id) {
{{ product.name }} - ${{ product.price }} (qty: {{ product.quantity }})
}

Spread Example - Apply 10% Discount:

@for (product of applyDiscount(products(), 10); track product.id) {
{{ product.name }} - ${{ product.price | number: "1.2-2" }} (qty: {{ product.quantity }})
}

applyDiscount(items, 10) - Uses spread: ...p, price: ...

Combined: Arrow Functions + Spread

Use arrow functions with spread operator in computed expressions:

Total Price: ${{ totalPrice() | number: "1.2-2" }}

(click)="products.update(items => applyDiscount(items, 15))" - Combines signal.update() with arrow function and spread inside method

Key Features:

  • Arrow Functions (v21.2): Write count.update(n => n + 1) directly - no wrapper method needed
  • Spread in Templates (v21.1): Use spread operator ...obj and arrays [...arr] in expressions
  • Signal Updates: Use signal.update(fn) with arrow function
  • Computed Values: Automatically recompute when dependencies change
