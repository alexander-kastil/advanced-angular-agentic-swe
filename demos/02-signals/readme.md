# Mastering Signals

Signals are Angular's primary reactive primitive for synchronous state management, replacing BehaviorSubjects and manual change detection triggers. This module covers the full signals API including `signal()`, `computed()`, `effect()`, `linkedSignal()`, `model()`, `input()`, and `output()`. You will learn the Container-Presenter pattern with signals, RxJS interop via `toSignal()`, and zoneless change detection.
